# h_basic_utils

A dart package for many helper methods fitting different situations.

## Install

### pubspec.yaml

Update pubspec.yaml and add the following line to your dependencies.

```yaml
dependencies:
  h_basic_utils: ^0.0.6
```

1. math_tils
   1. 四舍五入保留n位小数
   2. 获取最高位数+1 后的整数
   3. 摄氏度转华氏度
   4. 制冷量KW转RT
2. verify_utils
   1. 手机号验证
   2. 邮箱验证
   3. 字符串8~20包含大小写验证
3. string_utils
   1. 包含解决字符串省略提前太多问题
   2. md5加密
   3. 计算Text的size
4. time_utils
   1. 常用的时间方法的工具栏
